# Instruções de Teste (IT) - WeON: Minha Carteira

Este diretório contém as Instruções de Teste detalhadas para a funcionalidade **Minha Carteira** do produto **WeON**. As ITs foram estruturadas para garantir que todas as validações de UI/UX, regras de negócio e integrações de dados sejam cobertas conforme especificado pelo design.

## Estrutura dos Testes

O escopo foi dividido nos seguintes módulos funcionais:

1. **`01_KPIs_Cards.md`**: Validação dos totalizadores superiores (Contatos Totais, Ativos, Avaliação e Valor).
2. **`02_Graficos_e_Dashboards.md`**: Validação das visualizações gráficas de Risco vs. Valor e Engajamento vs. Valor.
3. **`03_Tabela_e_Filtros.md`**: Validação de listagem, paginação, pesquisa, filtros e abas de segmentação de contatos.

## Como utilizar
- **Designers / QA:** Utilizem os passos descritos como "Checklist" para homologar a entrega dos desenvolvedores.
- **Desenvolvedores:** Consultem as validações de "Resultado Esperado" para entender as regras esperadas nos componentes, formatações (R$, %, ⭐) e estados.
